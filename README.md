# MY-TV — Free IPTV Player

> **Your TV, Your Rules.**

MY-TV is a lightweight desktop IPTV player for Windows. Stream your own live channels, movies, series, and radio stations — all from a single, clean interface.

> ⚠️ **MY-TV is a media player only.** We do not provide playlists, channels, or any streaming content. You must bring your own M3U/M3U8 playlist from a provider of your choice.

## Getting Started

1. Download the latest release from the [Releases](../../releases) page.
2. Install and open MY-TV.
3. Add your M3U/M3U8 playlist URL or file.
4. Start watching.
